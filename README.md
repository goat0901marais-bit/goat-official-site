# GOAT Official Website

GOATの公式Webサイト用リポジトリです。  
撮影・採寸・出品代行・梱包・保管・発送までを一気通貫で支援する  
EC運営・物流サービスの紹介サイトを管理しています。

## Overview
本サイトは、GOATが提供するサービス内容、料金体系、会社情報、
お問い合わせ導線を掲載した静的Webサイトです。
